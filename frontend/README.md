# 🧠 Chanakya AI

**Chanakya AI** is an AI-powered goal execution platform that helps users transform ambitions into achievements through personalized roadmaps, actionable todo plans, psychological success systems, habit-building frameworks, and AI-powered guidance.

---

## 🚀 Features

### 🎯 Goal Management

* Create and manage personal goals
* Define goal type (Daily, Weekly, Monthly, Yearly, Lifetime)
* Track progress toward completion
* View goal-specific dashboards

### 🗺️ AI Roadmap Generator

Generate a complete roadmap for any goal.

Includes:

* Current situation analysis
* Strengths and weaknesses
* Skill gap identification
* Multi-phase execution plan
* Milestones
* Success metrics
* Risk management strategies

### ✅ AI Todo Generator

Automatically generates:

* Daily tasks
* Weekly tasks
* Milestone tasks
* Focus areas
* Top priorities
* Success metrics
* Recommended schedules

### 🧠 Psychological Success System

Creates a personalized execution system based on behavioral psychology.

Includes:

* Success Identity
* Non-Negotiable Rules
* Daily Execution System
* Habit Stacking Framework
* Environment Design
* Deep Work System
* Accountability System
* Low Motivation Protocol
* Failure Recovery System
* Long-Term Sustainability Plan
* Success Dashboard

### 💬 AI Goal Chat

Interactive AI assistant that helps users:

* Solve goal-related problems
* Clarify roadblocks
* Improve execution
* Stay focused on priorities

### 📊 Progress Tracking

Track:

* Goal completion
* Habit consistency
* Milestones achieved
* Focus hours
* Productivity trends

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/chanakya-ai.git
cd chanakya-ai
```

### Backend Setup

```bash
cd backend

npm install
```

Create `.env`

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

OPENROUTER_API_KEY=your_openrouter_key
```

Run backend:

```bash
npm run dev
```

---

### Frontend Setup

```bash
cd frontend

npm install
```

Create `.env`

```env
VITE_API_URL=http://localhost:5000
```

Run frontend:

```bash
npm run dev
```
