# 🚀 Chanakya AI

AI-powered Goal Achievement Platform that transforms goals into actionable Roadmaps, Todo Plans, and Success Systems using Generative AI.

---

## 📌 Overview

Chanakya AI helps users turn ambitious goals into structured execution plans.

Users can:

* Create goals
* Generate AI Roadmaps
* Generate AI Todo Plans
* Generate Success Systems
* Track progress
* Chat with AI for guidance

Inspired by Chanakya's strategic thinking and modern AI capabilities.

## 🚀 Local Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/chanakya-ai.git

cd chanakya-ai
```

### Backend Setup

```bash
cd backend

npm install

npm run dev
```

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

---

## 🌐 Deployment

### Backend (Render)

1. Push code to GitHub
2. Create Render Web Service
3. Connect GitHub repository
4. Set Root Directory:

```bash
backend
```

5. Build Command

```bash
npm install
```

6. Start Command

```bash
npm start
```

7. Add Environment Variables

```env
MONGODB_URI
JWT_SECRET
OPENROUTER_API_KEY
```

---

### Frontend (Vercel)

1. Import repository
2. Set Root Directory:

```bash
frontend
```

3. Add:

```env
VITE_API_URL=https://your-render-url.onrender.com
```

4. Deploy
